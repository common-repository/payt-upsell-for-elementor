This is a elementor add on for adding the upsell feature of Payt payment platform.

== Description ==

= Sobre o Plguin =
O plugin Payt Upsell e Downsell para Elementor é um add-on simples e fácil de usar para criar botões de upsell em suas páginas. Com esta extensão, você pode facilmente adicionar botões de upsell a qualquer página ou postagem do seu site com o Elementor.

= É Simples =
Basta arrastar e soltar o widget do botão de Upsell em sua página, personalizar as configurações de acordo com as instruções do seu Upsell Payt e tudo pronto para vender seu upsell!

Este plugin de upsell para Elementor é altamente personalizável e oferece uma variedade de opções de estilo para ajudá-lo a adaptar o botão às cores e estilo da sua oferta. Você pode escolher entre diferentes formatos, tamanhos e cores de botão, bem como adicionar texto personalizado e ícones.

= Objetivo =

O objetivo deste plugin de botão de upsell para Elementor é aumentar a conversão e simplicidade na implementação. Com sua facilidade de uso e recursos personalizáveis, você pode criar botões de upsell mais atraentes em momentos, sem a necessidade de conhecimentos em programação ou design.

== Frequently Asked Questions ==

= How do use it? =

Basta arrastar o elemento para sua página através da edição do Elementor.
Após adicionado a sua página, insira o código Upsell na Aba Elementor da sua edição de upsell na Payt.
